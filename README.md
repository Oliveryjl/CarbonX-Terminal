🌿 CarbonX Elite v9.5 | Decentralized Carbon Terminal
"Bridging the gap between blockchain theory and environmental finance practice."

📖 Project Overview
CarbonX Elite is an institutional-grade prototype developed as part of the G0253 Blockchain Finance course at Xiamen University Malaysia. This terminal simulates the full lifecycle of a carbon asset, specifically optimized for the Malaysia Bursa Carbon Exchange (BCX).The project addresses critical inefficiencies in traditional carbon markets, such as double counting, lack of transparency, and low liquidity. By leveraging blockchain as a "Single Source of Truth," we demonstrate how Digital MRV and AMM-based trading can transform voluntary carbon markets (VCM).

✨ Core Features
Institutional Onboarding: Secure node initialization for environmental entities.
Global Market Oracle: Real-time price tracking for Malaysia BCX, EU ETS, and China CCER.
Advanced dMRV Simulation: Automated biomass scanning using satellite data for verified token minting.
Atomic Swap Engine: Decentralized exchange simulation with instant liquidity for carbon tokens.
Proof of Burn: Permanent retirement protocol with generated on-chain certificates to prevent double counting.
Multi-Modal Interface: Seamless switching between English, 中文, and Bahasa Melayu, with adaptive Dark/Light themes.
