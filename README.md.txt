# 🌿 CarbonX Elite v9.5 | Institutional Carbon Terminal
**A Decentralized Carbon Asset Management & dMRV Protocol Prototype**

## 📖 Project Overview
CarbonX Elite is a high-fidelity terminal designed for institutional carbon credit management. It simulates the entire lifecycle of a carbon asset: from satellite-based **dMRV verification** to **AMM-based trading** and permanent **on-chain retirement**.

## ✨ Core Features
- **Institutional Gateway:** Secure node initialization and access control.
- **Global Market Integration:** Real-time price tracking for **Malaysia BCX** and EU-ETS markets.
- **Digital MRV:** Automated biomass scanning simulation using satellite oracle data.
- **Atomic Swap Engine:** Decentralized exchange simulation with instant liquidity.
- **Proof of Burn:** Generation of immutable on-chain retirement certificates.
- **Multilingual Support:** English, 中文, and Bahasa Melayu interfaces.

## 🛠️ Tech Stack
- **Engine:** Vanilla JavaScript (ES6+), HTML5, CSS3
- **Visualization:** Chart.js 4.0
- **UI Design:** Glassmorphism & Adaptive Themes (Dark/Light)
